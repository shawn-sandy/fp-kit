# Buttons


### Default Button

```html preview
<button
  type="button"
>
  Default Button
</button>
```

### Disabled Button

```html preview
<button
  type="button"
  disabled
>
  Disabled Button
</button>
```

### Red Button

Change the button color using the style attribute.

```html
style="--btn-bg:red; --btn-color:white;"
```

```html preview
<button
  type="button"
  style="--btn-bg:red; --btn-color:white;"
>
  Red Custom Style
</button>
```

#### Red Button (Disabled)

```html preview
<button
  type="button"
  disabled
  style="--btn-bg:red; --btn-color:white;"
>
  Red Disabled
</button>
```

### Large Button

```html
style="--btn-fs:var(--btn-lg);"
```

```html preview
<button
  type="button"
  style="--btn-fs:var(--btn-lg);"
>
  Large Button
</button>
```
