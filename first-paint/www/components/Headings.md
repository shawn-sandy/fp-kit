## Headings Variables

### Custom variables

```css
--h1: var(--fs-10);
--h2: var(--fs-9);
--h3: var(--fs-8);
--h4: var(--fs-7);
--h5: var(--fs-6);
--h6: var(--fs-5);
```

### Custom properties (rules)

```css

h1, h1[role="decoration"] {
  font-size: var(--h1);
}

h2, h2[role="decoration"] {
  font-size: var(--h2);
}

h3, h3[role="decoration"] {
  font-size: var(--h3);
}

h4, h4[role="decoration"] {
  font-size: var(--h4);
}

h5, h5[role="decoration"] {
  font-size: var(--h5);
}

h6, h6[role="decoration"] {
  font-size: var(--h6);
}
```
